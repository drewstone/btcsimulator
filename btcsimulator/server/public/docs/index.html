Getting started

  • Install Brunch: npm install -g brunch.
  • Install Bower: npm install -g bower.
  • Install Grunt: npm install -g grunt-cli.
  • Install project dependencies: npm install.
  • Install Bower components: bower install
  • Watch the project with continuous rebuild by brunch watch --server. This will also launch HTTP server on port 3333.
  • Build the minified project with brunch build -P.
  • Open the app to see the result.
  • Open the docs in /docs
  • Write your code in app dir, put scripts in vendor dir and write tests in tests.

Running the tests

  • Go to the test page
  • Or install mocha-phantomjs and phantomjs: npm install -g mocha-phantomjs phantomjs
