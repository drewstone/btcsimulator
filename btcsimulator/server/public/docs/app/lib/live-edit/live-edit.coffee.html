live-edit.coffee

app/lib/live-edit/
¶

This class will receive a model a will display all the fields defined in the live property in a way that when the user clicks the element an editor is displayed. This editor might be of the following types:

  1. Text input
  2. Select.

Create bindings for Backbone.Stickit


  createBindings: (liveData) ->
    @bindings = {}
    _.each liveData, (field) =>
      binding = observe: "#{field.name}"
      binding.format = @createCompositeFormat if field.composite
      @bindings["#live-#{field.cid}-#{field.name}"] = binding
